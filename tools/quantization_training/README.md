# Quantization Training Tools

Python scripts for training custom quantization models (AfricaQuant).

## Planned Scripts

- `train_africa_quant.py` - Train 1-bit quantization on African language models
- `fine_tune_swahili.py` - Fine-tune models on Swahili/English/Sheng data
- `export_gguf.py` - Export quantized models to GGUF format
- `benchmark_quantization.py` - Benchmark quantization performance

## Requirements

- Python 3.8+
- PyTorch
- transformers
- llama.cpp Python bindings

Scripts will be added as development progresses.
