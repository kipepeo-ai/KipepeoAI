## Technical Details

### Architecture
- **Core:** C/C++ libraries with ARM NEON optimizations
- **LLM Engine:** llama.cpp with AfricaQuant (1.58-bit quantization)
- **Video:** AV1 encoding/decoding with custom rate control
- **UI:** Jetpack Compose with Material 3

### Models Included (Default: 34B)
- Quantization: Q4_K_M (4-bit)
- Context length: 4096 tokens
- Languages: Swahili, English, Sheng
- Fine-tuned for Kenyan context
